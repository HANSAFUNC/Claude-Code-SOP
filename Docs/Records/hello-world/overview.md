# Hello World 示例模块

## 概述

Hello World 示例是一个最简单的 HarmonyOS ArkTS 声明式 UI 演示页面，用于学习和演示 ArkTS 基础语法。

## 模块路径

- 页面代码：`entry/src/main/ets/pages/hello/Index.ets`
- 测试代码：`entry/src/test/HelloWorld.test.ets`

## 技术栈

- HarmonyOS SDK
- ArkTS 语言
- ArkUI 声明式框架
