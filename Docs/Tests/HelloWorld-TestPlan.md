## 2. 测试用例

### 用例 1: 页面加载测试
- **目的**: 验证 HelloWorld 页面能正常加载
- **步骤**:
  1. 启动应用
  2. 通过路由访问 `pages/HelloWorld`
- **预期结果**: 页面无报错，正常渲染

### 用例 2: UI 显示测试
- **目的**: 验证文本内容正确显示
- **步骤**:
  1. 访问 HelloWorld 页面
  2. 检查页面文本
- **预期结果**:
  - 显示 "Hello World" (32 号字体，加粗)
  - 显示 "欢迎使用 HarmonyOS" (16 号字体)

### 用例 3: 路由测试
- **目的**: 验证路由配置正确
- **步骤**:
  1. 从其他页面通过 router.pushUrl 跳转
- **预期结果**: 能成功跳转到 HelloWorld 页面
