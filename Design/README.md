# UI 设计稿 (Design)

## 目录结构说明

本目录存放所有 UI 设计稿和原型图，按以下格式命名：

```
YYYYMMDD-{产品类型}-{功能模块}-{设计稿类型}.png
```

### 设计稿类型

- `wireframe` - 线框图
- `mockup` - 高保真设计图
- `flow` - 交互流程图
- `icon` - 图标资源

### 示例

- `20260325-保险培训平台 - 首页-mockup.png`
- `20260325-保险培训平台 - 登录页-wireframe.png`
- `20260326-保险培训平台 - 课程详情-flow.png`
